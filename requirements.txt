DADOS

Bases de dados do site kaggle: https://www.kaggle.com/code/allanbruno/helping-regular-people-price-listings-on-airbnb/input?select=total_data.csv

INSTALAÇÕES

pip install pandas
pip install seaborn
pip install plotly
pip install joblib
pip install streamlit


SOBRE INSTALAÇÕES

Instalações joblib e streamlit foram realizadas no Anaconda Prompt no diretório base do usuário.

SOBRE IDE

Projeto construído no Jupyter Notebook do Anaconda.

SOBRE DEPLOY

Efetive o deploy, no terminal Anaconda Prompt, entre na pasta do projeto e rode "streamlit run deploy.py" sem as aspas.
